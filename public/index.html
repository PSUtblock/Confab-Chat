    About The Application

    This application was made by Storm Crozier and Travis Block For Fall 2022 Full-Stack Class at Portland State University.

    The goal of the project was to build a web application using socket.io. the word 'Confab' is defined as: "an informal private conversation or discussion". So, in this instance the application envisioned was a chat and voice application for friends to connect and talk privately. So long as a user has the know-how to launch an instance of this, they and their friends can talk amongst themselves privately without worry.

    The style of the site is to provide a 'vintage' operating system look where modern styling practices are to a degree ignored. Looking back in a sense to this design is a sort of rebellion to what many cookie cutter themed sites of the modern era that concede to using generic templates. Somehow, the bland styles of the past have more character have more character than the future.
